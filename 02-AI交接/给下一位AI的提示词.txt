最高优先级硬约束：

- 固定代理及五个启动参数不可删除、替换、绕过、轮换或建议关闭；代理失效必须失败关闭。
- 不使用隔离 user-data-dir；继承原 Edge Cookie、登录、扩展、历史和权限。
- 这是 Android → Windows Edge 的真人实时网页控制器，不是爬虫、自动提问、回答抓取、批量操作或反检测浏览器。
- ChatGPT/Claude 使用严格人工模式：真实桌面 Edge 身份、真实窄窗口、鼠标/滚轮/键盘默认、原生触摸显式临时开启。
- 严格模式空闲不得长期启用 Runtime、DOM 或文件选择拦截；环境审计和上传必须由用户明确触发。
- 标签跟随默认 Windows UI Automation；不确定时保持当前标签；严格模式不得偷偷回退 document.hasFocus 轮询。
- 校准只画在手机控制页，不注入目标网站 DOM。
- 禁止指纹伪装、stealth、webdriver 隐藏、随机人类轨迹、验证码/限制绕过、自动发送和内容抓取。

v6.8.0 新增能力的边界（同样是硬约束）：

- 静止画面高清化（UX-008）：只用 Page 域被动截图补拍无损 PNG，不执行页面脚本、不启用 Runtime/DOM、不激活或置顶窗口、不注入内容、不循环；失败不进入重启恢复。
- 电脑剪贴板桥（CLIP-001/002）：只在用户当次显式点击时读/写 Windows 剪贴板，单次、双向，经 PowerShell 在浏览器之外实现，不轮询、不自动同步、不落盘、不解析、不记录正文，超限拒绝不截断。
- 手机专用窗口（WIN-001/002）：可选、默认关闭；同 Edge 配置新开独立窄窗口，不改动主窗口；标签面板与 UIA 跟随只作用于专用窗口，集合外标签不切换，歧义仍保持当前标签。

一条统领原则：任何体验优化，先问“会不会被网页 JS 观测到、是否需要激活窗口或注入内容”；能在操作系统层/合成器层完成的绝不放进目标网页。

明确永久否决（不得以体验为名重新引入）：DOM 镜像渲染、为观感激活窗口、自建 WebRTC/视频编码、系统音频转发、反检测/stealth/指纹伪装/随机人类行为。

工程：仓库为唯一事实源，交接材料只在 02-AI交接/ 保留一份，发布 ZIP 的 AI交接/ 由 scripts/package-release.mjs 复制生成；ZIP 与 SHA-256 由 GitHub Actions 在打 tag 时产出。
