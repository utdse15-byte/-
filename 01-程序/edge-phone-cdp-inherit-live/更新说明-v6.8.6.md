# v6.8.6 更新说明

发布日期：2026-07-20

v6.8.6 是对 v6.8.5 的两轮深度审查修复版：一轮内部对抗审计（实时同步状态机 / 画面档 / 输入法与连发三路并行），一轮第三方外部审计（6 项发现全部核实属实）。没有新功能，全部是把已有功能的正确性边界补严。所有硬约束（固定代理及五个启动参数、继承原 Edge 配置、严格人工模式、禁止自动化/抓取/伪装、"零页面信号"边界）保持不变。

## 一、令牌轮换即撤销（外部审计 P1）

此前"轮换访问令牌"只能阻止旧令牌**新建**连接：已经在线的旧连接不受影响，仍可点"接管控制"并继续注入点击、输入、读取剪贴板——与界面"轮换后旧令牌立即失效"的承诺不符。轮换本来就是在怀疑链接泄漏时使用的，恰恰在这个场景下失效。本版引入**认证代数**：

- 连接建立时盖上当前代；轮换时代数自增，除发起方外的所有连接一律以 4003 关闭；
- 服务端消息入口按代数丢弃旧代的一切在途数据（文本命令、触摸、二进制上传块），命令处理在 claimControl 之前再查一层；
- 手机端收到 4003 后清除已失效的本地令牌并停驻（复用双开停驻机制），提示用电脑上的新链接/新令牌重新进入，不做注定失败的自动重连；
- 新增双客户端回归测试：轮换后旧连接被 4003 关闭、发起方保持有效、旧令牌拒新建、新令牌可连接。

## 二、标签身份判定（外部审计 P1 + 设计缺陷）

- **地址归一化不再合并语义不同的标签页**：`view-source:` 保留为身份前缀（CDP 视源代码页为独立标签）；fragment 保留（hash 路由的单页应用靠它区分页面）；按 URI 规范只小写 scheme/host，路径/查询保留大小写。协议省略、www、结尾斜杠等对地址栏可访问性文本的模糊宽容不变，前缀匹配边界补 `#`；
- **专用窗口跟随改为"先全集匹配、后作用域检查"**：此前先把专用窗口之外的标签从候选里删掉再匹配，会把"前台其实是主窗口里同 URL 的标签"（按设计应保持不动）错判成专用窗口内的唯一命中而误切换。现在先在全部标签里判定前台身份，跨集合内外同 URL 视为歧义、判定结果在集合外则保持当前标签——过滤不再制造假唯一；
- 新增回归：三组地址非碰撞断言、省略 hash 的前缀命中、集合内外同 URL 歧义保持不动、集合内独有 URL 照常跟随。

## 三、实时同步防误删加固（内部审计）

v6.8.5 的实时同步依赖"远程光标停在已同步文本末尾"这一前提。审计找出多条打破前提却未暂停同步的路径，全部堵上：

- **同 URL 刷新与 SPA 前进后退**不改变页面标识、异步失效检测漏判——后退/前进/刷新/跳转网址现在同步作废基准；
- **切换/关闭标签页**先同步作废基准，防止 pageState 滞后期间一次防抖差量落进另一个标签页的输入框；
- **在途竞态**：差量任务在每个远程写入之间按"代"自查，中途有失效事件（点画面/切页/断线/失败）立刻停手；
- **并发外部编辑**：破坏性整段替换前先一次性只读校验远程全文仍等于基准（桌面端或页面脚本改过就拒绝替换并要求重新取回）；
- **回车发送**不再假设输入框已清空：发送后重新取回对齐——聊天框变空、多行文本域保留文字（不丢稿），取不回则保守失效；
- 空基准失效也真正暂停同步；（重）连接后作废断线前的旧基准。

## 四、画面与按键

- **自动档回归控制端压力**：v6.8.5 曾改为取所有手机的最坏链路，会被一台卡住/半死的只读手机把整机画质永久压成幻灯片（每台手机本就有独立的最新帧限流保护）。全局质量应服务于正在操作的控制端；
- **清晰档探测退避**：因过载从清晰档降下来后压住探测 3 分钟，消除"升清晰→过载→降均衡→又探清晰"每 20 秒一次的画面闪烁抖动；
- 长按连发的重复 click 抑制改用 `detail===0` 识别键盘/无障碍激活：既不双发，也不误伤键盘用户。

## 五、已确认、列入下版专项的架构债（外部审计 P2/P3）

以下三项经核实属实，但属于有分量的重构，单独成版配专门测试轮，不混入本批：

1. Edge 历史查询同步阻塞主事件循环（代码内有自知注释）——应移入 Worker/子进程；
2. 历史库文件复制快照非原子——应改为 Worker 内稳定读事务/一致备份（注：快照回退存在的原因正是 Edge 锁库，此场景下 SQLite Backup API 同样可能打不开源库，实现须以重试兜底）；
3. 电脑文件目录同步遍历——应移入 Worker 或异步 API + 有界并发 + 可取消。

对"一台手机、家庭局域网"的日常使用，这三项触发条件有限；随历史库和目录规模增长会逐渐明显。

真机验收边界与 v6.8.x 各版相同，详见 `发布验证报告.md`；历史版本修复清单见对应更新说明。
